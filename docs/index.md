# eodh-workflows

Workflows for EO DataHub platform.

## Getting started

Create the environment:

```shell
make env
```

Run `pre-commit` hooks:

```shell
make pc
```

## Docs

To build project documentation run:

```shell
make docs
```

and then:

```shell
mkdocs serve
```
